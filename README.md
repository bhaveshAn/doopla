===============================
doopla
===============================
H(ad)oopla!
A Python script to fetch the output of fail tasks from ours Hadoop clusters

Features
--------
* Automatically get the last failed job for a user
* Code highlighting via `Pygments`.

Install via pip
------
Install v

Requirements
------------
- Python >= 2.6 or >= 3.3
